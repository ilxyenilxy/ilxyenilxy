안녕하세요. 저는 AI에 대해 공부하고있으며, 주로 Computer Vision에 대해 다루었습니다. <br>
<br>
AI 모델의 아키텍쳐를 구성하고 성능을 끌어올리는 데에 집중하며 최근에는 모델의 경량화와 배포 그리고 모델을 운영하는 프로세스에 새로운 배움의 기회를 잡고있습니다. 때로는 fusion, inventor와 같은 모델링 툴로 시제품을 설계하고 제작하거나 MCU를 이용하여 하드웨어 라인을 구축하고 하드웨어 디바이스를 혼용하여 IOT 프로젝트에 보다 하드웨어에 가깝게 개발하기도 합니다.

#### 🖥️ AI
주로 TensorFlow를 사용하며, CNN, TransFormer, LSTM과 같은 알고리즘을 이용해 이미지와 영상에 대해 처리 및 추론할 수 있는 모델을 설계하고 학습시키는 일을 해왔습니다. 또한, Jetson과 NPU와 같은 Edge Device에서 해당 모델을 최적화하고 실시간으로 구동될 수 있게 포팅하며 모델이 구동되는 환경인 하드웨어에 대한 지식도 쌓으려 노력하고 있습니다.

#### 📱 Edge & Embedded Systems
IOT 서비스에 관한 개발을 주로 진행했습니다. Arduino 기반 MCU와 Linux 기반 Edge 디바이스를 다룰 수 있으며, C++ 그리고 CV를 활용하여 하드웨어 디바이스에 판단력을 심어주는 노력을 하고 있습니다.

#### ☁️ Infra, Data (Studying)
모델이 실질적인 서비스에서 어떻게 배포되며 어떻게 관리되는 지에 대해 궁금증을 가지고 공부 하고있습니다. 실제 운영되는 서비스에서 모델과 배포의 단계를 동시에 확인하고 벤치바킹하여 해당 모델이 배포 상황에서 최적의 효율과 속도를 구현할 수 있도록 하는 것이 목표입니다.
